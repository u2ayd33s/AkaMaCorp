## 気づき
エージェントディレクトリに `01_`, `02_` 等の番号を付けると、ファイルシステム上で役割の優先順位が視覚的に明確になる。

## 適用場面
新しいエージェントを追加する際、番号を決めることで組織内の位置づけを意識できる。
