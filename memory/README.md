# AkaMaCorp エクスペリエンス層 (Experiential Layer)

育成優先型開発 (Nurture-First Development) におけるエクスペリエンス層。
日常の対話・作業から得た知識・洞察・エラーパターンを蓄積し、定期的に結晶化してスキル層へ昇格させる。

## ディレクトリ構成

```
memory/
├── README.md              ← このファイル（使い方説明）
├── insights.md            ← 重要な洞察・発見のログ [INSIGHT]
├── error-patterns.md      ← エラー・失敗パターンの記録 [ERROR]
├── daily/                 ← 日次ログ（YYYY-MM-DD.md）
│   └── 2026-03-19.md
└── crystallization/       ← 結晶化チェックポイントの記録
    └── CC1-2026-03-19.md
```

## ログ記録ルール

### タグ体系
- `[DECISION]` — 行った決断・選択とその理由
- `[INSIGHT]` — 重要な気づき・発見・パターン認識
- `[ERROR]` — 失敗・ミス・誤解とその修正原則
- `[PATTERN]` — 繰り返し見られる行動・要求パターン
- `[TRACE]` — 推論過程の記録（なぜその判断をしたか）

### 日次ログの書き方
`daily/YYYY-MM-DD.md` に追記形式で記録する:

```markdown
## YYYY-MM-DD

### [DECISION] タイトル
- 決断内容: ...
- 理由: ...
- 結果: ...

### [INSIGHT] タイトル
- 気づき: ...
- 適用場面: ...

### [ERROR] タイトル
- 何が起きたか: ...
- 修正した方向: ...
- 再発防止原則: ...
```

## 結晶化のトリガー

以下のいずれかで結晶化を実施する:
- **スケジュール**: 毎月16日（月次レビュー）
- **閾値**: 未結晶化ログが20件を超えたとき
- **イベント**: 重要なプロジェクト完了・新エージェント追加時

結晶化は `nfd-crystallize` スキルを使って実施する。
