# 豊田市バス時刻表サイネージ — CSV書き出しルール

> 基本ルールは調布駅と共通。[[chofu-station-signage/csv-rules]] を先に読むこと。
> 豊田市固有の差異のみここに記載する。

## 基本仕様（調布駅と共通）

| 項目 | 値 |
|------|----|
| ファイル名 | `{シート名}.csv`（例: `豊田市①.csv`） |
| 保存先 | `data/csv/` |
| エンコーディング | **UTF-8（BOMなし）** |
| 区切り文字 | カンマ `,` |
| 改行コード | LF推奨 |

## 対象シート

> 初回チェック時に確定すること。

```
豊田市①、豊田市①_en
豊田市②、豊田市②_en
豊田市③、豊田市③_en
豊田市⑤、豊田市⑤_en
豊田市駅東口、豊田市駅東口_en
```

> `_en` ファイルが存在することから英語版CSVも管理対象。
> 差分チェックでの英語版の扱いは初回チェック時に確認・記載すること。
